Resumen de Elementos de Formulario

  • text: Permite introducir texto simple (ejemplo: placeholder, required, autofocus).
  • checkbox: Selección múltiple de opciones.
  • email: Campo para ingresar correos electrónicos con validación básica (ejemplo: placeholder, required).
  • tel: Campo para números de teléfono (ejemplo: pattern).
  • select: Desplegable de selección única.
  • color: Selector de color.
  • range: Selector de rango numérico deslizante (ejemplo: min, max, step).
  • date: Selector de fecha (ejemplo: required).
  • time: Selector de hora.
  • url: Campo para URL, validado como dirección web (ejemplo: placeholder).
  • search: Campo de búsqueda que añade un icono específico en algunos navegadores.
  • number: Campo numérico, con opción de min, max, step.
  • datetime-local: Fecha y hora en formato local (ejemplo: required).
  • button: Botón para ejecutar acciones en JavaScript.
  • file: Campo para subir archivos (ejemplo: multiple, required).
  • hidden: Valor oculto que no se muestra en el formulario.
  • image: Botón de envío que usa una imagen.
  • password: Campo para contraseñas, enmascarado (ejemplo: required).
  • radio: Selección única entre varias opciones.
  • reset: Botón para restablecer el formulario.
  • submit: Botón para enviar el formulario.
  • datetime: Fecha y hora en formato UTC (no soportado en todos los navegadores).
  • month: Selector de mes.
  • week: Selector de semana.

Descripción de Atributos de Formularios en HTML5
Atributo Descripción
placeholder Muestra un texto de ejemplo en el campo de entrada que desaparece al escribir.
required Obliga al usuario a completar el campo antes de enviar el formulario.
pattern Define una expresión regular que debe cumplir el valor ingresado.
min / max Establece los valores mínimo y máximo para los campos numéricos y de rango.
step Define el incremento de valor en campos numéricos y de rango.
multiple Permite seleccionar múltiples opciones, como archivos en un campo de tipo file.
autofocus Coloca el cursor en el campo automáticamente al cargar la página.
value Define un valor preestablecido en el campo de entrada.
readonly Hace que el campo sea de solo lectura, sin permitir cambios.
disabled Desactiva el campo, evitando que se pueda usar o enviar.
size Determina el ancho del campo de entrada en caracteres.
maxlength Establece la cantidad máxima de caracteres permitidos en el campo.
