Guía de Elementos Semánticos en HTML5

Breve descripción y ejemplos de uso de las etiquetas semánticas de HTML5.

Elemento Descripción Ejemplo
<s> Representa contenido que ya no es relevante o aplicable, mostrándolo con un tachado. Este artículo ya no está disponible.
<ins> y <del> Indican inserciones y eliminaciones en el documento. <ins> subraya el texto añadido y <del> tacha el texto eliminado. Reunión: Jueves Viernes
<q> Define una cita breve en el texto, con comillas automáticas. Esta es una cita breve.
<abbr> y <dfn> Para abreviaturas y definiciones. <abbr> define el texto abreviado y <dfn> destaca una definición. HTML es el lenguaje de marcado de la web.
<code> Para mostrar código de programación en el documento. let x = 10;
<samp> Representa la salida de un sistema informático o aplicación. Error: Archivo no encontrado
<kbd> Indica combinaciones de teclas o entradas del usuario. Para guardar, presione Ctrl + S.
<var> Representa una variable en un contexto matemático o de programación. x = 10
<data> Asocia contenido con un valor legible por máquina. diez
<sup> y <sub> Representan superíndice y subíndice, utilizados para convenciones tipográficas. H2O y E = mc2
