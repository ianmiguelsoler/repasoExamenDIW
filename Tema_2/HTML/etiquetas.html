Etiquetas HTML5

Una guía con explicación y ejemplos para las etiquetas HTML5 destacadas.

<header>

Define una cabecera para un documento o sección, incluyendo logotipos o menús.

<header>Contenido de cabecera</header>

<nav>

Representa una sección de navegación con enlaces.

<nav><a href="#">Inicio</a></nav>

<section>

Define una sección de contenido temático.

<section>Contenido de la sección</section>

<aside>

Elemento para contenido relacionado, como barras laterales.

<aside>Contenido adicional</aside>

<footer>

Define el pie de página del documento o sección.

<footer>Información del pie</footer>

<article>

Representa contenido independiente como un artículo de blog.

<article>Contenido del artículo</article>

<figure> y <figcaption>

Contenedor para contenido ilustrativo, con una leyenda descriptiva.

<figure>
    <img src="imagen.jpg" alt="Descripción">
    <figcaption>Leyenda de la imagen</figcaption>
</figure>

<address>

Elemento para información de contacto.

<address>Dirección de contacto</address>

<bdi>

La etiqueta <bdi> (Bidirectional Isolation) permite aislar un fragmento de texto que puede tener una direccionalidad diferente o desconocida.

                <p>Usuario: <bdi>إيان</bdi> </p>

Resultado:

Usuario: إيان

<cite>

La etiqueta <cite> se utiliza para citar el título de una obra, como un libro, película, o una obra de arte.

                <p>Me encanta <cite>Don Quijote de la Mancha</cite> de Cervantes.</p>

Resultado:

Me encanta Don Quijote de la Mancha de Cervantes.

<datalist>

Elemento para listas de autocompletado en inputs.

<input list="navegadores">
<datalist id="navegadores">
    <option value="Chrome">
    <option value="Firefox">
</datalist>

<details> y <summary>

Permiten mostrar u ocultar detalles adicionales al hacer clic.

<details>
    <summary>Mostrar más</summary>
    Texto adicional visible al pulsar.
</details>

<embed>

La etiqueta <embed> se usa para integrar contenido externo, como un vídeo o archivo de audio, en el documento.

                <embed src="sample-video.mp4" width="400" height="300">

<main>

Elemento que contiene el contenido principal del documento.

<main>Contenido principal</main>

<mark>

Resalta texto importante en un contexto específico.

<mark>Texto resaltado</mark>

<meter>

Elemento para representar una medida en una escala acotada.

<meter value="0.7">70%</meter>

<output>

Define el resultado de un cálculo.

<form oninput="resultado.value=parseInt(a.value)+parseInt(b.value)">
    <input name="a" type="number"> + <input name="b" type="number">
    = <output name="resultado"></output>
</form>

<progress>

Representa el progreso de una tarea en proceso.

<progress value="70" max="100">70%</progress>

<ruby>, <rp>, y <rt>

La etiqueta <ruby> se usa para anotaciones en textos de idiomas ideográficos (como japonés o chino). El texto de la anotación se coloca dentro de <rt>, y <rp> ofrece soporte para navegadores que no muestran la anotación.

                <ruby>
                    漢 <rt>kan</rt>
                    字 <rt>ji</rt>
                </ruby>

Resultado:

漢 kan 字 ji

<small>

La etiqueta <small> se utiliza para mostrar texto que es menos importante, como notas legales o aclaraciones.

                <p>Derechos reservados <small>© 2024</small></p>

Resultado:

Derechos reservados © 2024

<time>

Representa una fecha o una hora en formato legible.

<time datetime="2024-10-30">30 de octubre de 2024</time>

<wbr>

La etiqueta <wbr> (Word Break Opportunity) permite definir un posible punto de salto de línea en una palabra larga.

                <p>XML<wbr>Http<wbr>Request</p>

Resultado:

XMLHttpRequest
