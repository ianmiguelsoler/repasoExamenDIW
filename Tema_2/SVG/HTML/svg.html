Introducción a SVG en HTML5

SVG (Scalable Vector Graphics) es un formato para crear imágenes vectoriales en dos dimensiones, compatible con HTML5 y otros entornos web.

1. La Etiqueta SVG

SVG es una tecnología basada en XML que permite crear gráficos vectoriales bidimensionales. A diferencia de <canvas>, SVG es independiente de la resolución y cada elemento gráfico pertenece al DOM, por lo que se puede manipular con JavaScript o CSS.

SVG admite gráficos geométricos, imágenes bitmap y texto, permitiendo agrupación, transformación y aplicación de estilos comunes a los elementos.

2. Creación de una Imagen con SVG

Para crear una imagen SVG, necesitamos definir un sistema de coordenadas en el que el origen es la esquina superior izquierda.

Podemos crear un documento SVG utilizando la estructura:

<svg width="200" height="200">
  <!-- Contenido de la imagen SVG -->
</svg>

Este código puede ser insertado directamente en un documento HTML o referenciado como una imagen externa con la etiqueta <img> o <iframe>.

3. Ejemplo de SVG en HTML5: Dibujando un Círculo

El elemento <circle> se utiliza para dibujar un círculo. A continuación se explican sus atributos principales:

  • cx y cy: Coordenadas x e y del centro del círculo. Si se omiten, el centro estará en el origen (0, 0).
  • r: Define el radio del círculo (ejemplo: 40px).
  • stroke: Color del borde del círculo.
  • stroke-width: Grosor del borde.
  • fill: Color de relleno del círculo (ejemplo: #6AB150).

Ejemplo de código para crear un círculo:

<svg width="250" height="100">
  <circle cx="125" cy="50" r="40" stroke="#030" stroke-width="4" fill="#6AB150" />
</svg>

Uso de la Etiqueta <defs>

La etiqueta <defs> en SVG se utiliza para definir elementos gráficos que no se muestran directamente en el lienzo, sino que se almacenan para ser reutilizados en otras partes del SVG. Esto es útil para definir gradientes, patrones, filtros, máscaras, y otros elementos gráficos que pueden aplicarse mediante una referencia.

Cuando defines un elemento dentro de <defs>, necesitas asignarle un id que lo identifique. Luego, puedes referenciar este elemento en otras partes del SVG usando url(#id).

Ejemplo de Uso de <defs> con un Gradiente Lineal

En el siguiente ejemplo, el gradiente lineal se define dentro de <defs> con el identificador miGradiente. Luego, se aplica como relleno de un rectángulo usando fill="url(#miGradiente)". 

<svg width="200" height="200">
  <defs>
      <linearGradient id="miGradiente" x1="0%" y1="0%" x2="100%" y2="0%">
          <stop offset="0%" style="stop-color:blue; stop-opacity:1;" />
          <stop offset="100%" style="stop-color:red; stop-opacity:1;" />
      </linearGradient>
  </defs>
  <rect x="10" y="10" width="180" height="180" fill="url(#miGradiente)" />
</svg>

Resultado:

En este ejemplo, el rectángulo se rellena con el gradiente lineal definido en <defs>, aplicando una transición de color de azul a rojo.

Ejemplo de Línea en SVG

El elemento <line> se utiliza para dibujar una línea recta entre dos puntos. Los atributos x1 y y1 definen el punto de inicio, mientras que x2 y y2 definen el punto final de la línea. Puedes ajustar el color y el grosor de la línea con los atributos stroke y stroke-width.

Ejemplo de código:

      <svg width="200" height="100">
        <line x1="10" y1="10" x2="190" y2="90" stroke="blue" stroke-width="4" />
      </svg>

8. Creación de Gradientes en SVG

En SVG, podemos crear degradados de color lineales y radiales utilizando los elementos <linearGradient> y <radialGradient>, respectivamente. Los degradados permiten una transición suave entre colores, útil para aplicar efectos visuales atractivos.

Gradiente Lineal

El siguiente ejemplo muestra un degradado lineal con transiciones entre varios colores usando el elemento <linearGradient>: 

<svg width="250" height="100">
  <defs>
      <linearGradient id="linearGradient">
          <stop offset="10%" stop-color="red" />
          <stop offset="30%" stop-color="blue" />
          <stop offset="50%" stop-color="green" />
          <stop offset="70%" stop-color="yellow" />
          <stop offset="90%" stop-color="red" />
      </linearGradient>
  </defs>
  <rect fill="url(#linearGradient)" width="250" height="100" />
</svg>

Resultado:

Gradiente Radial

El siguiente ejemplo muestra un degradado radial usando el elemento <radialGradient>, donde el color cambia desde el centro hacia el borde de la figura: 

<svg width="250" height="150">
  <defs>
      <radialGradient id="radialGradient">
          <stop offset="0%" stop-color="chartreuse" />
          <stop offset="100%" stop-color="#030" />
      </radialGradient>
  </defs>
  <rect fill="url(#radialGradient)" width="250" height="150" />
</svg>

Resultado:

En estos ejemplos, el atributo offset en <stop> define la posición donde un color específico comienza o termina en el gradiente. Los valores van de 0% (inicio del gradiente) a 100% (final del gradiente).

9. Patrones de Relleno en SVG

En SVG, se pueden crear patrones de relleno personalizados utilizando la etiqueta <pattern>. Esto permite repetir una imagen o forma en un área específica, como el relleno de un rectángulo o cualquier otra figura.

Los patrones de relleno se definen dentro de <defs> y se les asigna un id para ser referenciados. Luego, pueden aplicarse como relleno de otros elementos mediante fill="url(#id)".

Ejemplo de Patrón de Relleno con una Imagen

En el siguiente ejemplo, se define un patrón de relleno utilizando una imagen dentro de <pattern>. Este patrón se aplica luego al relleno de un rectángulo. 

<svg width="250" height="200">
  <defs>
      <pattern id="imagenPatron" width="125" height="131" patternUnits="userSpaceOnUse">
          <image xlink:href="https://via.placeholder.com/125x131" width="125" height="131" />
      </pattern>
  </defs>
  <rect width="250" height="200" fill="url(#imagenPatron)" />
</svg>

Resultado:

Ejemplo de Patrón de Relleno con una Forma

A continuación se muestra un ejemplo en el que el patrón de relleno está formado por elipses repetidas. Este patrón se aplica a un rectángulo. 

<svg width="250" height="120">
  <defs>
      <pattern id="elipsePatron" x="5" y="0" width="40" height="20" patternUnits="userSpaceOnUse">
          <ellipse cx="20" cy="10" rx="20" ry="10" fill="#6AB150" />
      </pattern>
  </defs>
  <rect x="25" y="10" width="200" height="100" fill="url(#elipsePatron)" stroke="#f00" />
</svg>

Resultado:

En estos ejemplos, el atributo patternUnits="userSpaceOnUse" define cómo se coordina el patrón en el área de relleno. Además, podemos ajustar el tamaño y posición de repetición con los atributos width y height del patrón.
